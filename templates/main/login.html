Instructions for Master Trainers

  • Select User type → Master Trainer; if you are new click Sign up.
  • Enter a valid mobile number and create a secure password.
  • Log in using your mobile number (username) and the password you created.
  • From the navbar open New submission request → Create new. Fill in details carefully and upload certificates for the appropriate themes/modules.
  • Only one submission request is allowed per Master Trainer.
  • After submitting, monitor your dashboard for reviewer feedback. If a certificate is reverted, upload the corrected certificate and follow any reviewer instructions.
  • Keep your mobile number active — reviewers may contact you for clarifications.
